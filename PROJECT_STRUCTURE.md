## Directory Explanations

*   **WebGL/**: Contains experiments and tutorials related to WebGL.
*   **attempt2/ to attempt8/**: A series of directories, likely representing different stages or versions of a canvas-based project, as documented in the original `README.md`.
*   **langton3d/**: An experiment implementing Langton's Ant in a 3D environment.
*   **langtonHexD/**: An experiment with Langton's Ant on a hexagonal grid.
*   **langtonSound/**: An experiment that adds sound to the Langton's Ant simulation.
*   **utils/**: Contains utility JavaScript files, such as `aframe.min.js` and a custom canvas library.
*   **root directory**: Contains the main `README.md`, license, and other configuration files. The `TUTORIAL.md` file contains the original content of the `README.md`, and this `PROJECT_STRUCTURE.md` file provides this overview.
